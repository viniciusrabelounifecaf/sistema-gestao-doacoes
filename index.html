Objetivo do sistema

  • Registrar as doações recebidas pela comunidade.
  • Consultar os itens disponíveis em estoque.
  • Registrar as entregas realizadas às famílias atendidas.
  • Manter um histórico simples das movimentações.

Observação sobre o MVP

Este sistema é um protótipo funcional local. Os dados ficam salvos apenas no navegador utilizado para a demonstração, sem exposição pública de informações.

Para preservar a privacidade, recomenda-se utilizar identificações simples, como "Família 01" ou "Beneficiário 01", sem cadastrar documentos pessoais.
