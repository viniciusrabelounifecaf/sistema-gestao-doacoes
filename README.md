# Sistema de Gestão de Doações

Este projeto é um MVP de um sistema web desenvolvido para apoiar o Projeto Social Ação do Bem na organização das doações recebidas, no controle de estoque e no registro das entregas realizadas.

A ideia surgiu a partir da necessidade de centralizar informações que antes eram registradas de forma manual ou dispersa, facilitando a consulta dos itens disponíveis e o acompanhamento das movimentações.

## Funcionalidades

- Cadastro de doações recebidas
- Registro do tipo de item, quantidade, unidade de medida e origem da doação
- Controle automático dos itens disponíveis em estoque
- Registro de entregas realizadas às famílias atendidas
- Histórico de entradas e saídas
- Painel inicial com indicadores gerais
- Opção para limpar os dados usados em testes

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Observação sobre os dados

Este sistema é um protótipo funcional local. Os dados ficam salvos apenas no navegador utilizado, por meio do armazenamento local do navegador.

Para preservar a privacidade, recomenda-se utilizar identificações simples, como "Família 01" ou "Beneficiário 01", sem cadastrar documentos pessoais.

## Finalidade

O projeto foi desenvolvido para fins acadêmicos, como parte de uma atividade extensionista do curso de Análise e Desenvolvimento de Sistemas.

O objetivo principal é demonstrar como uma solução simples pode contribuir para melhorar a organização de processos sociais, especialmente em iniciativas comunitárias de pequeno porte.
